### 方案 1：一键转换（1小时）

在销售订单列表页面添加：

```python
# 前端按钮
<el-button
  type="primary"
  @click="convertToWorkOrder(row)"
  :disabled="row.converted">
  {{ row.converted ? '已转换' : '转换为施工单' }}
</el-button>

# API 调用
convertToWorkOrder(salesOrder) {
  this.$post('/api/v1/workorders-flow/create_from_sales_order/', {
    sales_order_id: salesOrder.id
  })
}
```

**效果**：
- 点击即可转换
- 5 分钟完成
- 立即见效
