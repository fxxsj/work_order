## 👥 用户角色和权限

### 角色定义

#### 1. 系统管理员
- **权限**：所有系统功能
- **职责**：系统维护、用户管理、全局配置
- **主要操作**：
  - 创建和管理用户账号
  - 配置系统参数
  - 管理全局权限

#### 2. 业务员
- **权限**：施工单创建、客户管理、基础查询
- **职责**：客户沟通、施工单创建、进度跟踪
- **主要操作**：
  - 创建新施工单
  - 管理客户信息
  - 查看生产进度

#### 3. 生产经理
- **权限**：生产计划、任务分派、进度管理
- **职责**：生产调度、资源分配、质量控制
- **主要操作**：
  - 分派生产任务
  - 监控生产进度
  - 管理工序流程

#### 4. 部门主管
- **权限**：本部门任务管理、员工管理
- **职责**：部门内部管理、任务协调
- **主要操作**：
  - 管理本部门员工
  - 分配部门任务
  - 审批部门工作

#### 5. 操作员
- **权限**：任务执行、状态更新
- **职责**：具体工序操作、质量检查
- **主要操作**：
  - 执行分配的任务
  - 更新工作状态
  - 报告工作完成

#### 6. 质检员
- **权限**：质量检查、验收管理
- **职责**：质量监控、问题发现
- **主要操作**：
  - 质量检查和记录
  - 问题报告和跟踪
  - 验收确认

## 🛠️ 核心功能操作

### 1. 施工单管理

#### 创建施工单

**步骤1：基本信息录入**
```
1. 进入"施工单管理"页面
2. 点击"新建施工单"按钮
3. 填写基本信息：
   - 施工单号：系统自动生成或手动输入
   - 客户选择：从下拉菜单选择或新建
   - 业务员：默认当前登录用户
   - 优先级：低/正常/高/紧急
```

**步骤2：产品和工序配置**
```
1. 添加产品信息：
   - 产品名称：选择或新建产品
   - 规格说明：详细的产品规格
   - 数量：生产数量
   - 单价：产品单价

2. 配置工序：
   - 选择需要的工序
   - 设置每个工序的负责人
   - 确定时间计划
```

**步骤3：审核和提交**
```
1. 检查所有信息完整性
2. 确认价格和数量
3. 提交审核（如果需要）
4. 保存施工单
```

#### 施工单状态管理

**状态流转图**：
```
待开始 → 进行中 → 已完成
   ↓        ↓        ↓
已暂停    已取消   已完成
   ↓
已取消
```

**状态变更操作**：
```
1. 在施工单详情页面
2. 点击"更改状态"按钮
3. 选择新状态
4. 添加变更说明（可选）
5. 确认状态变更
```

### 2. 任务管理

#### 查看任务列表

**筛选选项**：
```
- 按状态筛选：待开始/进行中/已完成/已暂停
- 按负责人筛选：特定用户
- 按时间筛选：开始日期/截止日期
- 按优先级筛选：低/正常/高/紧急
```

**视图切换**：
```
- 列表视图：详细信息表格
- 看板视图：拖拽式任务板
- 甘特图：时间线和依赖关系
```

#### 任务操作

**开始任务**：
```
1. 选择"进行中"的任务
2. 点击"开始工作"按钮
3. 确认开始时间
4. 更新任务状态
```

**完成任务**：
```
1. 填写完成数量
2. 记录次品数量
3. 上传完成照片（如需要）
4. 添加完成说明
5. 点击"标记完成"
```

### 3. 客户管理

#### 客户信息录入

**基本信息**：
```
- 客户名称：完整的公司名称
- 联系人：主要联系人姓名
- 联系方式：电话、邮箱
- 地址：详细的收货地址
```

**业务信息**：
```
- 信用等级：A/B/C等级
- 付款方式：月结/现结/预付
- 业务员：负责该客户的业务员
```

#### 客户关系维护

**历史记录查看**：
```
1. 选择客户
2. 查看"历史施工单"
3. 分析合作历史
4. 评估客户价值
```

### 4. 物料管理

#### 物料入库

**入库操作流程**：
```
1. 进入"物料管理"页面
2. 点击"物料入库"
3. 选择物料类型
4. 录入数量和单价
5. 扫描条码或手动输入
6. 确认入库信息
```

**库存监控**：
```
- 当前库存：实时显示
- 安全库存：低于设定值时预警
- 库存周转：分析使用频率
```

#### 物料出库

**出库流程**：
```
1. 选择要使用的物料
2. 输入出库数量
3. 关联到具体施工单
4. 确认出库操作
5. 更新库存记录
```

## ❓ 常见问题

### Q1: 忘记密码怎么办？

**解决方案**：
```
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱或用户名
3. 查找重置密码邮件
4. 点击邮件中的重置链接
5. 设置新密码
6. 使用新密码登录
```

### Q2: 施工单无法创建？

**可能原因和解决**：
```
原因1：权限不足
解决：联系管理员分配相应权限

原因2：必填信息缺失
解决：检查所有必填字段，确保信息完整

原因3：网络连接问题
解决：检查网络连接，重试操作
```

### Q3: 任务无法分配？

**排查步骤**：
```
1. 检查用户权限
2. 确认用户状态
3. 检查任务状态
4. 验证部门设置
5. 联系系统管理员
```

### Q4: 数据导出失败？

**解决方法**：
```
1. 检查网络连接
2. 减少导出数据量
3. 更换浏览器
4. 清除浏览器缓存
5. 联系技术支持
```

### Q5: 页面加载缓慢？

**优化建议**：
```
1. 检查网络速度
2. 清除浏览器缓存
3. 使用推荐浏览器（Chrome/Firefox）
4. 减少同时打开的标签页
5. 联系IT部门检查服务器状态
```

## 🎯 最佳实践

### 1. 数据录入规范

#### 施工单创建
```
✅ 正确做法：
- 完整填写客户信息
- 准确记录产品规格
- 及时更新生产进度
- 规范填写备注信息

❌ 错误做法：
- 留空必填字段
- 使用模糊的产品描述
- 延迟更新任务状态
- 填写无意义的备注
```

#### 任务管理
```
✅ 正确做法：
- 及时开始分配的任务
- 准确记录完成数量
- 上传相关图片文档
- 及时报告异常情况

❌ 错误做法：
- 忽略任务截止时间
- 估算错误的完成时间
- 遗漏质量检查记录
- 不及时更新任务状态
```
