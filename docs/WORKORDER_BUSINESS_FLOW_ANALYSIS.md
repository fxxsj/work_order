## 一、整体业务流程概览

```
创建施工单
    ↓
填写基本信息（客户、产品、数量等）
    ↓
选择工序（根据产品默认工序或手动选择）
    ↓
选择版（图稿、刀模、烫金版、压凸版等，根据工序要求）
    ↓
提交审核（业务员审核）
    ↓
审核通过 → 自动变更状态为"进行中"
    ↓
开始工序 → 自动生成任务 → 自动分派到部门/操作员 ✅
    ↓
部门协作完成任务（可手动调整分派）
    ↓
任务完成 → 自动判断工序是否完成
    ↓
所有工序完成 → 自动标记施工单为完成 ✅
```

#### 2.4.2 自动分派机制 ✅ **已实现（2026-01-07）**

**已实现：**
- ✅ 任务生成时自动分派（`_auto_assign_task()` 方法）
- ✅ 优先使用工序级别的分派（`WorkOrderProcess.department` 和 `WorkOrderProcess.operator`）
- ✅ 如果工序未指定部门，根据部门-工序关联关系自动分派
- ✅ **智能分派规则**：优先匹配专业车间，排除外协车间，最后选择父部门

**部门分派规则（优先级从高到低）：**
1. **工序级别分派（最高优先级）**：如果工序已指定部门（`WorkOrderProcess.department`），直接使用
2. **专业车间匹配（高优先级）**：如果工序编码与部门编码匹配，优先选择该专业车间
   - `DIE` (模切) → `die_cutting` (模切车间)
   - `PRT` (印刷) → `printing` (印刷车间)
   - `CUT` (开料) → `cutting` (裁切车间)
   - `PACK` (包装) → `packaging` (包装车间)
   - `CTP` (制版) → `design` (设计部)
3. **非外协专业车间（中优先级）**：排除外协车间，选择其他专业车间（子部门），按 `sort_order` 排序
4. **外协车间（低优先级）**：如果没有专业车间可选，选择外协车间
5. **父部门（兜底）**：如果以上都没有，选择父部门（如生产部）

**操作员分派规则：**
- ✅ 如果工序已指定操作员（`WorkOrderProcess.operator`），任务继承该操作员
- ⚠️ 否则，任务的操作员为空（`assigned_operator = None`），需要后续手动分派
- 📝 **未来扩展**：可以根据部门、任务类型、工作量等智能分派操作员（代码中已预留扩展点）

```python
        status='completed'
    ).count() == 0
    
    if all_processes_completed and work_order.status != 'completed':
        work_order.status = 'completed'
        work_order.save()
```

```python
@action(detail=True, methods=['post'])
def update_quantity(self, request, pk=None):
```
